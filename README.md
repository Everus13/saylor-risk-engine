# MSTR-BTC Impact & Execution Engine

Система предиктивной аналитики и оптимального исполнения (Optimal Execution) для расчета долговых обязательств MicroStrategy (MSTR) и оптимизации продаж биткоина с использованием машинного обучения (Quantile Regression) и обучения с подкреплением (RL).

## Структура проекта
* `config/` — конфигурационные файлы (календарь долгов).
* `src/` — исходный код бизнес-логики:
  * `config.py` — настройки и гиперпараметры.
  * `financial_tracker.py` — расчет обязательств MSTR.
  * `order_book_sim.py` — симулятор стакана.
  * `ml_impact.py` — прогнозирование ценового влияния.
  * `rl_agent/` — среда и обучение RL-агента.
* `app.py` — интерактивный Streamlit дашборд.
